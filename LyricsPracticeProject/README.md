# LyricsPracticeProject

LyricsPracticeProject — учебное Windows Forms приложение на C#, которое работает со сторонними REST API.

## Назначение проекта

Приложение позволяет искать песни по названию или исполнителю, выбирать найденную песню из списка и получать текст песни.

## Используемый стек

- C#
- Windows Forms
- .NET Framework 4.8
- HttpClient
- Newtonsoft.Json
- REST API

## Используемые API

### 1. iTunes Search API

Используется для поиска песен по названию или исполнителю.

Пример endpoint:

https://itunes.apple.com/search?term=coldplay&entity=song&limit=10

### 2. lyrics.ovh API

Используется для получения текста песни по исполнителю и названию.

Пример endpoint:

https://api.lyrics.ovh/v1/Coldplay/Yellow

## Основные возможности

- поиск песен через iTunes API;
- вывод списка найденных песен;
- автоматическое заполнение исполнителя и названия песни;
- получение текста песни через lyrics.ovh API;
- обработка ошибок при пустом запросе или отсутствии текста;
- очистка формы.

## Структура проекта

- Models — классы для хранения данных из JSON-ответов;
- Services — классы для работы с API и форматирования текста;
- Form1 — основная форма приложения.

\# Структура проекта



\## Models



Содержит классы, которые описывают данные из JSON-ответов API.



\## Services



Содержит классы для выполнения HTTP-запросов, обработки JSON и форматирования текста песни.



\## Form1



Основная форма приложения. Отвечает за взаимодействие с пользователем: ввод запроса, выбор песни, получение текста и вывод статуса.

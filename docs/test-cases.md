\# Тестовые сценарии



| № | Действие | Ожидаемый результат |

|---|----------|---------------------|

| 1 | Оставить поле поиска пустым и нажать поиск | Появляется сообщение о необходимости ввести запрос |

| 2 | Ввести исполнителя и нажать поиск | Отображается список найденных песен |

| 3 | Выбрать песню из списка | Поля исполнитель и название заполняются автоматически |

| 4 | Нажать "Найти текст" | В текстовом поле отображается текст песни |

| 5 | Нажать "Очистить" | Все поля формы очищаются |
